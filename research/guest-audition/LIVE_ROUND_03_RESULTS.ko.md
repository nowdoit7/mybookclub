# 3차 실제 API 오디션 결과 — 두 책 교차 검증

날짜: 2026-07-20

모델: GPT-5.6

제품 런타임 변경: 없음

## 방법

2차에서 만든 파스칼·뉴턴·애덤 스미스의 자료 기반 묘사와 말투 지문을 실제
GPT-5.6 생성에 사용했습니다. 사용자가 이미 완주한 『불변의 법칙』과 『삼체』
대화에서 책 근거, 논점, 사용자 주장, 발언 위치를 동결했습니다. 각 인물은 두 책에
한 번씩 들어가 총 6개의 게스트 샘플을 만들었습니다. 비교 대상은 같은 문맥의 기존
일반 독자 발언입니다.

세 명의 새로운 평가자는 인물 정체와 어느 샘플이 게스트인지 모른 채 두 샘플을
채점했습니다. 점수·선호·근거를 잠근 뒤에만 정체를 공개하여 가짜 인용, 허위 전기,
현대 책을 실제로 읽었다는 주장, 유명세를 이용한 권위, 말투 과사용, 다른 책 정보
유출을 검사했습니다.

기록된 최종 실행은 6호출, 합산 45.5초였고 개별 호출은 6.0~10.3초였습니다. 그 전
교정 실행은 3건 생성 후 네 번째 응답이 출력 한도에서 잘렸고, 당시 하네스가 성공
결과를 즉시 저장하지 않는 문제도 드러났습니다. 이후 낮은 추론 예산, 1회 제한 복구,
호출별 체크포인트, 유료 결과 덮어쓰기 방지를 추가했습니다. 따라서 이 작업에서 발생한
최상위 API 요청은 교정 4회와 기록 실행 6회, 총 10회입니다.

## 실제 생성 대사에서 확인한 점

### 파스칼

『불변의 법칙』에서는 다른 독자의 우려를 인정한 뒤 “현재 자원을 덜 쓰는 사람”과
“실패했을 때 회복할 수 없게 되는 사람”을 나눠 보자는 질문으로 이동했습니다. 기존
일반 독자보다 책에 구체적이고 자연스럽다는 평가가 세 명 모두에게서 나왔습니다.

그러나 영문 『삼체』에서는 한국어로 저장된 말투 도입 예시를 그대로 따라 두 발언을
모두 한국어로 생성했습니다. 이는 묘사 가설의 실패라기보다 데이터 설계의 명확한
실패입니다. 말투 지문은 완성 문장 예시가 아니라 언어 중립적인 수사 동작과 언어별
표현층으로 분리해야 합니다.

### 뉴턴

『불변의 법칙』에서는 귀환하지 못한 비행기의 증거가 무한한 대비를 요구하지는
않는다고 선을 긋고, 준비로 줄일 수 있는 위험에 자원을 우선하자는 구체적 구분을
더했습니다. 2차의 기계적인 방법론 과시는 줄었고 세 평가자가 모두 선택했습니다.

반면 『삼체』에서는 “관찰과 결론을 나누자”는 시작이 자연스러웠지만, 기존 일반
독자가 이미 월페이서·뤄지·장베이하이·웨이드의 구체적 장면을 더 날카롭게 연결하고
있었습니다. 유명 인물을 넣었다는 사실만으로 강한 일반 독자를 이기지 못했습니다.

### 애덤 스미스

『불변의 법칙』에서는 안전의 값을 누가 내는지 묻고 현재 기회의 손실까지 함께
계산하여 2차의 일반론 문제를 줄였습니다. 세 평가자가 모두 게스트를 선택했습니다.

『삼체』에서도 공정성과 취소 가능한 권한을 자연스럽게 물었지만, 새로운 해석은 기존
일반 독자보다 1.33점 낮았습니다. 스미스의 렌즈가 틀린 것은 아니지만 해당 토론에서
필요한 새 압력을 만들지는 못했습니다.

## 평가

이번 결과는 “유명인은 재미있다”보다 중요한 세 가지 조건을 보여줍니다.

1. **장르보다 현재 논점의 친화도가 중요합니다.** 『불변의 법칙』의 누락된 위험,
   안전마진, 부담 분배는 세 인물의 사고 렌즈가 모두 구체적으로 작동했습니다.
2. **강한 일반 독자를 대체할 때는 이름이 품질을 보장하지 않습니다.** 『삼체』의 기존
   반론은 이미 장면 근거와 논쟁 압력이 높아 세 게스트가 모두 블라인드에서 졌습니다.
3. **말투 예문을 데이터로 저장하면 다국어 누출이 생깁니다.** 의미상의 수사 동작과
   영어·한국어 실현을 분리하지 않으면 세션 언어 고정 규칙을 깰 수 있습니다.

『불변의 법칙』에서 세 명이 모두 통과했다는 결과도 세 사람 모두를 구현하라는 뜻이
아닙니다. 동일한 위험 논점이 세 렌즈에 우호적이었고 기준선이 상대적으로 약했을
가능성이 큽니다. 인물 고유성보다는 “잘 맞는 논점에 역사적 사고 렌즈를 넣으면 대화가
좋아질 수 있다”는 것까지만 확인했습니다.

## 결정

- 제품 런타임과 `SPEC.md`는 계속 변경하지 않습니다.
- 파스칼은 선두 후보지만, 언어 중립 지문과 언어별 표현층으로 바꾼 뒤 영문 교차
  테스트를 다시 통과해야 합니다.
- 뉴턴과 스미스는 탈락시키지 않되 **책 장르**가 아니라 **추출된 토론 논점과의
  친화도**를 등장 조건으로 시험해야 합니다.
- 다음 유료 실험 전에는 각 후보마다 성격이 다른 친화 도서 한 권을 더 추가하고,
  의도적 비궁합 도서에서는 게스트가 등장하지 않는 코드 판단을 먼저 무료로 검증합니다.
- 역사적 게스트는 일반 독자보다 낫다고 예측되는 드문 경우에만 같은 범주의 독자 한
  명을 교체해야 합니다. 매 세션 등장하거나 네 번째 인물로 추가하지 않습니다.
