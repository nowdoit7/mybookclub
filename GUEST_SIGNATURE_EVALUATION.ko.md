# 상상 속 게스트 특징 발화 평가

평가일: 2026-07-21  
모델: GPT-5.6  
대상: 역사적 인물, 전설적 인물, 문학 속 인물 21명

## 평가 목적

상상 속 게스트가 일반 독자처럼만 말하지 않고, 한 세션에 한 번은 자신의 업적이나 대표적인 사고 습관에서 비롯된 특징을 자연스럽게 드러내는지 확인한다. 특정 유명 도서에 맞춘 문장을 테스트하지 않기 위해 안전, 책임, 불완전한 증거를 다루는 중립적인 가상 도서 《경계의 기록》을 모든 게스트에게 동일하게 제공했다.

## 판정 기준

- 현재 책에 대한 실제 해석을 더 깊게 만드는가?
- 인물의 이름을 가려도 다른 게스트와 구별되는 사고 습관이나 이미지가 있는가?
- 이력서, 역사 강의, 유명세를 근거로 한 권위 주장처럼 들리지 않는가?
- 실제 인용처럼 꾸미거나 상상 속 인물이라는 설정을 대사에서 노출하지 않는가?
- 2~4문장 규칙과 저작권 규칙을 지키는가?

## 1차 결과와 프롬프트 수정

1차 21인 호출은 모두 성공했고 모든 발화에 특징이 들어갔다. 다만 여러 게스트가 “제 독법으로 보면”, “살펴온 제 관점에서는”처럼 자신의 특징을 설명했고, 파스칼은 “상상 속 독자로서”라고 말해 몰입을 깼다.

그래서 특징을 설명하지 말고 현재 주장 안에서 직접 **수행**하도록 프롬프트를 수정했다. 이름난 업적이나 작품명은 현재 해석과 문법적으로 분리할 수 없을 만큼 자연스러울 때만 선택적으로 허용하고, 대사에서 상상 속 인물임을 다시 밝히는 것도 금지했다.
